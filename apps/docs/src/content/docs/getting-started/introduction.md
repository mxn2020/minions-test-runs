# Minions Test-runs

Execution records, turn-by-turn logs, and assertion results

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-test-runs/sdk`) — Core library for Node.js
- **Python SDK** (`minions-test-runs`) — Python bindings
- **CLI Tool** (`@minions-test-runs/cli`) — Command-line interface
- **Web App** — Interactive playground at [test-runs.minions.wtf](https://test-runs.minions.wtf)
